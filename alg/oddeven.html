A parallel-friendly Bubble variant: alternate passes comparing disjoint pairs (0,1),(2,3),… then (1,2),(3,4),… until no swaps.

Pseudocode


01swapped ← true; pass ← 0
02while swapped:
03  swapped ← false
04  if pass is even:  // even pass
05    for j = 0,2,4,.. < n-1:
06      if A[j] > A[j+1]: swap; swapped ← true
07  else:              // odd pass
08    for j = 1,3,5,.. < n-1:
09      if A[j] > A[j+1]: swap; swapped ← true
10  pass ← pass + 1

How It Works

Each iteration does two disjoint comparison passes (even then odd indices), similar to a synchronous version of Bubble. Stops when a full iteration makes no swaps.

  • Amber: currently compared pair.
  • Rose: a swap just occurred.
  • Emerald: final (fully sorted) state.

Complexity

Time
  • Worst: O(n²)
  • Average: O(n²)
  • Best: O(n) if already sorted
Space

O(1) extra memory (in-place)
