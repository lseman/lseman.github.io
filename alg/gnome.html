Gnome Sort — Interactive Visualizer

Gnome Sort (a.k.a. Stupid Sort) walks a pointer i forward when adjacent order is OK; if not, it swaps and steps back—like a gnome tidying flowers by moving one at a time.

Pseudocode


01i ← 1
02while i < n:
03  if A[i-1] ≤ A[i]:
04    i ← i + 1
05  else:
06    swap(A[i-1], A[i])
07    if i > 1: i ← i - 1 else i ← 1

How It Works

Think of insertion sort without shifting: move forward when adjacent order is correct; when out of order, swap and step back to re-check the previous pair.

  • Amber: compared pair (i-1, i).
  • Rose: a swap just occurred.
  • Emerald: final state (fully sorted).

Complexity
